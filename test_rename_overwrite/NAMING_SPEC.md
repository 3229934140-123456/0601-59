# my_project 命名规范

## 命名规则

- 使用小写字母和下划线 (snake_case)
- 格式: `{theme}_{name}_{index:03d}.{ext}`
- 示例: `hero_banner_001.jpg`

## 主题分类

- default

## 文件格式规范

### 图片
- 支持格式: .jpg, .jpeg, .png, .gif, .webp, .svg, .bmp, .tiff
- 推荐格式: PNG (透明背景), JPG (照片), WebP (网页)
- 最小分辨率: 1024x768

### 字体
- 支持格式: .ttf, .otf, .woff, .woff2
- 推荐格式: WOFF2, TTF

### 图标
- 支持格式: .svg, .png, .ico
- 推荐格式: SVG (矢量), PNG (位图)
